Booleans

Python có một kiểu dữ liệu là bool. Nó nhận hai giá trị là True hoặc False.

x = True
print(x)
print(type(x))

True
<class ‘bool’>

Thường chúng ta sẽ không trực tiếp sử dụng các giá trị True/False này, mà ta sẽ sử dụng kết quả của các phép toán Boolean. Chúng ta sẽ sử dụng những toán tử sau để làm việc với kiểu dữ liệu này.

Comparison Operations (So sánh)

Toán tử Mô tả
a == b a bằng b?
a < b a nhỏ hơn b?
a <= b a nhỏ hơn hoặc bằng b?
a != b a khác b?
a > b a lớn hơn b?
a >= b a lớn hơn hoặc bằng b? 
def can_run_for_president(age):
    """Can someone of the given age run for president in the US?"""
    # The US Constitution says you must be at least 35 years old
    return age >= 35

print("Can a 19-year-old run for president?", can_run_for_president(19))
print("Can a 45-year-old run for president?", can_run_for_president(45))

Can a 19-year-old run for president? False
Can a 45-year-old run for president? True

Thường các phép so sánh sẽ trả về kết quả đúng như ta dự đoán

3.0 == 3

True

Nhưng đôi khi kết quả lại không như bạn nghĩ

'3' == 3

False

Bạn có thể đoán tại sao kết quả lại ra False chứ? Gợi ý: hãy sử dụng hàm type để biết ‘3’ và 3 khác nhau như thế nào nhé.

Ta có thể sử dụng kết hợp các toán tử Boolean với toán tử số học mà ta đã được học ở bài trước để đánh giá bất kỳ một biểu thức toán học nào. Ví dụ, chúng ta có thể kiểm tra tính chẵn lẻ dựa trên việc so sánh phép toán chia lấy dư dư một số bất kỳ với 2:

def is_odd(n):
    return (n % 2) == 1

print("Is 100 odd?", is_odd(100))
print("Is -1 odd?", is_odd(-1))

Is 100 odd? False
Is -1 odd? True

Hãy nhớ rằng toán tử so sánh là “==”. Nếu bạn viết n == 2, bạn đang thực hiện so sánh số n có giá trị bằng 2 không. Khi bạn viết n = 2, bạn đang gán giá trị 2 cho biến n.

Combining Boolean Values (Phép toán với các giá trị boolean)

Ta có thể thực hiện các phép toán với các giá trị Boolean dựa trên khái niệm phép giao, phép hợp và phép đối trong tập hợp. Từ khóa của các phép này trong Python lần lượt là AND, OR và NOT.

Với các toán tử trên, chúng ta có thể khiến hàm can_run_for_president chạy chính xác hơn:

def can_run_for_president(age, is_natural_born_citizen):
    """Can someone of the given age and citizenship status run for president in the US?"""
    # The US Constitution says you must be a natural born citizen *and* at least 35 years old
    return is_natural_born_citizen and (age >= 35)

print(can_run_for_president(19, True))
print(can_run_for_president(55, False))
print(can_run_for_president(55, True))

False
False
True

Bạn có thể đoán nhanh kết quả của biểu thức Boolean dưới không?

True or True and False

Để trả lời cho câu hỏi này, chúng ta cần biết được thứ tự của các toán tử Boolean.

Ví dụ, lệnh AND có độ ưu tiên cáo hơn OR. Chính vì vậy ta sẽ thực hiện phép TRUE and False trước, nó có kết quả là False. Sau đó thực hiện phép True or False và kết quả cuối cùng của biểu thức là True.

Nếu ta thực hiện hiểu thức từ trái sang phải thì kết quả sẽ là gì? True of True = True và _True and False = False.

Ta luôn có thể ghi nhớ thứ tự ưu tiên của các phép toán, thế nhưng để chặt chẽ hơn, ta nên sử dụng dấu ngoặc.

Hãy nhìn vào ví dụ sau:

prepared_for_weather = have_umbrella or rain_level < 5 and have_hood or not rain_level > 0 and is_workday

Biểu thức này xem xét việc ta có bị kẹt nếu trời đổ cơn mưa không,

  • Nếu như ta có cái ô
  • Hoặc nếu như trời chỉ mưa nhẹ và ta có mũ
  • Nếu không, ta vẫn sẽ ổn, trừ khi trời mưa và đó là ngày thường

Thế nhưng, bạn có thể thấy đoạn code trên vừa khó đọc, mà còn có lỗi. Ta có thể giải quyết vấn đề này bằng cách thêm vài dấu ngoặc:

prepared_for_weather = have_umbrella or (rain_level < 5 and have_hood) or not (rain_level > 0 and is_workday)

Nếu cảm thấy vẫn khó đọc, ta có thể tiếp tục phân nhỏ lệnh trên bằng cách thêm vài dấu ngoặc nữa:

prepared_for_weather = have_umbrella or ((rain_level < 5) and have_hood) or (not (rain_level > 0 and is_workday))

Bây giờ bạn có thể đọc và hiểu dễ dàng hơn chứ? Ngoài ra ta còn có thể phân biểu thức trên thành nhiều hàng như sau:

prepared_for_weather = (
    have_umbrella 
    or ((rain_level < 5) and have_hood) 
    or (not (rain_level > 0 and is_workday))
)

Conditionals (Điều kiện)

Kiểu Boolean sẽ phát huy năng lực nhất khi được kết hợp với các cấu trúc điều kiện rẽ nhánh, sử dụng các từ khóa if, elif và else.

Các cấu trúc rẽ nhánh, thường gọi là các câu lệnh if – then, giúp ta kiểm soát được khối code này sẽ được thực thi dựa trên kết quả của các biểu thức Boolean. Dưới đây là một ví dụ:

def inspect(x):
    if x == 0:
        print(x, "is zero")
    elif x > 0:
        print(x, "is positive")
    elif x < 0:
        print(x, "is negative")
    else:
        print(x, "is unlike anything I've ever seen...")

inspect(0)
inspect(-15)

0 is zero
-15 is negative

Từ khoá if và else được sử dụng rộng rãi trong nhiều ngôn ngữ lập trình. Có một từ khoá thú vị ở đây là elif - nó được rút gọn từ else if. Trong cấu trúc rẽ nhánh thì khối lệnh elif và else là tuỳ chọn, có nghĩa là nó có thể xuất hiện hoặc không, còn khối lệnh if bắt buộc phải có. Ngoài ra thì ta có thể thêm bao nhiêu khối lệnh elif cũng được, nhưng chỉ được một khối lệnh else ở cuối cùng.

Chúng ta hãy chú ý cách Python sử dụng dấu hai chấm (:) và khoảng trắng để phân tách các khối lệnh tách biệt hẳn với nhau. Cách viết này giống như cách ta định nghĩa một hàm mới, câu lệnh khai báo hàm sẽ kết thúc với dấu hai chấm (:), và dòng tiếp theo được thụt lùi. Những dòng thụt lùi tiếp theo thuộc về khối code đó, cho đến khi ta gặp được dòng không thụt lùi đầu tiên.

def f(x):
    if x > 0:
        print("Only printed when x is positive; x =", x)
        print("Also only printed when x is positive; x =", x)
    print("Always printed, regardless of x's value; x =", x)

f(1)
f(0)

Only printed when x is positive; x = 1
Also only printed when x is positive; x = 1
Always printed, regardless of x’s value; x = 1
Always printed, regardless of x’s value; x = 0

Boolean conversion (Chuyển đổi)

Ta đã thấy chức năng ép kiểu, với hàm int() chuyển bất kỳ đối số nào thành int - hay còn gọi là số nguyên, và hàm float() chuyển tất cả đối số nào thành float - số thực. Nên có thể bạn sẽ không hề ngạc nhiên nếu như trong Python có tích hợp hàm bool() để biến các đối số truyền vào thành dạng Boolean.

print(bool(1)) # all numbers are treated as true, except 0
print(bool(0))
print(bool("asf")) # all strings are treated as true, except the empty string ""
print(bool(""))
# Generally empty sequences (strings, lists, and other types we've yet to see like lists and tuples)
# are "falsey" and the rest are "truthy"

True
False
True
False

Ta có thể sử dụng các biến không thuộc kiểu Boolean trong câu điều kiện if, và Python sẽ tự động thế các biến này với giá trị Boolean tương ứng chính xác:

if 0:
    print(0)
elif "spam":
    print("spam")

spam

Cuối cùng là lời cảm ơn sâu sắc tới Trương Quốc Quân - người đồng hành cùng mình trong series bài viết này.
